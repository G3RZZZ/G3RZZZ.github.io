Headers:

Elementos usados para dividir o distinguir secciones, hay 6 tamaños usualmente usados para establecer la relevancia del contenido. Los headers también son útiles para indexar la página.

Ejemplo:

  • Header 1

  • Header 2

  • Header 3

Primer Hello World!

Hello World!



p

Elemento que representa bloques de texto.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Tenetur eaque inventore ipsa aperiam at aspernatur eius, ab minus, cumque velit nobis aliquam natus commodi, perferendis doloremque saepe nisi voluptates. Voluptatem!



i

Elemento que contiene texto en itálica.

Ejemplo:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Ab facere accusantium quidem ratione tempora. Veritatis recusandae id aliquam. Vel, molestias. Corrupti iusto voluptatum quidem, corporis in molestias fuga laborum minus?



b

Elemento que pone el texto dentro de él en negritas.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Doloremque, voluptatum cum natus, illum quos dolores et placeat porro error accusantium quidem expedita, nam explicabo possimus corrupti? Veniam culpa eos vel!



strong

Elemento que resalta el texto que contiene, esto por su importancia por medio del uso de negritas.

Ejemplo:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Officiis exercitationem adipisci quas ipsam nihil asperiores, laborum, totam recusandae ipsum natus labore vel sequi distinctio. Dolor repudiandae repellat impedit? Nostrum, doloribus.



u

Elemento usado para subrayar texto dentro.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Illum inventore incidunt cumque velit debitis sequi reprehenderit ullam veritatis et qui iure quae, dolore at ducimus consequatur esse illo aliquam cupiditate!



strike

Elemento que tacha el texto horizontalmente.

Ejemplo:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolore quaerat quasi, porro rem temporibus culpa excepturi placeat ut itaque perferendis nihil nisi tempora harum tenetur laudantium optio commodi ratione quis?



small

Elemento que hace el texto dentro más pequeño por debajo de un nivel del usado.

Ejemplo:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Obcaecati laudantium culpa minima impedit rem nihil accusantium perspiciatis pariatur saepe, laborum minus libero ad molestiae debitis harum sed dolor vel deserunt.



em

Elemento que enfatiza el texto dentro, inclinándolo hacia la derecha y adelgazándolo un poco.

Ejemplo:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Doloremque accusantium expedita debitis beatae consequuntur cumque molestias dolorum omnis blanditiis iusto, recusandae autem, fugit asperiores itaque dolor assumenda quia ratione consectetur?



section

Elemento que divide un texto y header del resto del documento sin mayor propósito.

Ejemplo:

lorem

Lorem ipsum dolor sit amet consectetur adipisicing elit. Pariatur aliquam doloribus et. Et, adipisci ipsam pariatur perspiciatis nostrum cumque consectetur modi ullam. Eligendi eum excepturi, voluptas illum praesentium nesciunt voluptatum?



aside

Elemento que coloca el texto que contiene a un lado del que le sigue, el diseño del texto debe ser especificado con criterios de margin, padding, width, etc. en css.

Ejemplo:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Laborum, officiis aspernatur vitae fuga sit nemo praesentium rerum, reiciendis, numquam tenetur expedita architecto quaerat accusantium pariatur. Ducimus eius et facere soluta!

Lorem ipsum dolor sit amet consectetur adipisicing elit. (texto en aside)

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Suscipit maxime optio, tempora numquam magni, velit ipsum similique et reiciendis, soluta possimus saepe ipsa quidem libero amet porro eligendi animi molestias.



footer

Elemento cuyo texto contenido sirve como pie de página al contenido que le precede. Este footer puede ser diseñado en un css específicamente.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. (texto previo)

Lorem ipsum (footer)



nav

Elemento que secciona las partes (links) de una página actual para su navegación. Puede ser definida y diseñada en css.

Ejemplo:

siguiente pag
otra pag


Dentro de un div

Un elemento div funciona como un contenedor, no afecta el interior de ninguna manera hasta que se le agrega un style usando css.



br

Elemento que crea un salto de línea en el texto o sección en el que se coloque.

Ejemplo:

salto 1
salto 2
salto 3



Dentro del header

Elemento header, es representativo de contenido introductorio o navegación.



Span

El elemento de span funciona como contendor genérico de contenido, dicho contenedor puede tener id o class y ser diseñado con css posteriormente.



Article

Este elemento sirve para separar y contener información que sea distribuye y se reúsa, tal como una publicación o artículo de periódico.



Listas

Lista sin orden - ul

Elemento que declara una lista sin orden, se usa puntos por cada elemento listado.

Ejemplo:

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4

Listas con orden - ol

Elemento que declara una lista sin orden, se usa números por cada elemento listado.

Ejemplo:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3


Tablas

Elemento que tabula texto o información que contiene. Utiliza tr para las filas y td la información dentro de esa celda.

Ejemplo:

h1 h2 h3
r11 r12 r13
r21 r22 r23
r31 r32 r33
